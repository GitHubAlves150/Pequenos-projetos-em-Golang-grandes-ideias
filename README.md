# Pequenos-projetos-em-Golang-grandes-ideias
Este repositório tem como objetivo documentar as principais estruturas de dados em Golang.

A Pasta Generic_Junior_Pleno é uma coleção de exerciciso gradual que mostra como trabalhar com a técnica de métodos genéricos ou para quem está migrando do C++ "templates"
Templates é muito rico para reutilização de codigo quando é preciso utilizar dois ou mais tipos de dados para uma única operação
float64, int, double, string, struct
